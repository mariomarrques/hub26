# Tasks — Suppliers UI Polish

## Task 1 — Polir a página pública de fornecedores
### Objetivo
Melhorar layout, hierarquia visual e responsividade da página de fornecedores.

### Fazer
- Ajustar header da página
- Melhorar layout dos cards
- Melhorar espaçamento, alinhamento e densidade visual
- Garantir boa responsividade

### Pronto quando
- Página estiver mais limpa e legível
- Cards estiverem visualmente melhores
- Mobile estiver funcional e confortável

### Limites
- Sem mudar lógica de negócio
- Sem redesenho global da aplicação

---

## Task 2 — Polir modal de detalhes
### Objetivo
Melhorar a experiência de leitura e confiança dentro do detalhe do fornecedor.

### Fazer
- Reorganizar header do modal
- Criar agrupamento melhor das informações
- Destacar descrição
- Melhorar lista de avaliações

### Pronto quando
- Modal estiver mais claro
- Informações estiverem mais bem organizadas
- Reviews estiverem mais agradáveis de consumir

### Limites
- Sem mudar fluxo de dados
- Sem adicionar nova regra de negócio

## Task 3 — Transformar links em botões por tipo de rede
### Objetivo
Exibir links do fornecedor como botões visuais por rede/plataforma.

### Fazer
- Renderizar botões por tipo:
  - WhatsApp
  - Grupo
  - Instagram
  - Site
- Adicionar ícones correspondentes
- Aplicar estilo visual coerente por tipo
- Mostrar só quando existir link

### Pronto quando
- Bloco de links estiver visualmente forte
- Cada rede tiver identidade clara
- Não houver poluição visual

### Limites
- Sem excesso de cores
- Sem gradientes exagerados

---
